Create an extraordinary, fully responsive anime-themed birthday celebration website titled:

**“Mishu × Zeno — A Legendary Birthday Saga”**

The entire experience should be heavily inspired by the intense energy, cosmic atmosphere, transformations, battles, and celebratory spirit associated with Dragon Ball–style anime. It must feel like entering an interactive anime episode rather than visiting a normal birthday webpage.

Do not create a generic template or childish birthday page. Build a premium, cinematic, highly immersive website with powerful animations, energy effects, sound design, interactive surprises, dramatic typography, and smooth storytelling.

## Technology

Build the website using:

* React with Vite
* Tailwind CSS v4.2+
* Framer Motion for interface animations
* GSAP with ScrollTrigger for cinematic scroll sequences
* Howler.js or native Web Audio API for sound controls
* Lucide React icons
* Canvas or lightweight particle effects for energy particles
* Fully responsive design for mobile, tablet, laptop, and large desktop
* Optimized assets, lazy loading, and smooth performance
* Clean component-based architecture

## Motion Guidelines

Animations should feel intense but polished.

Use:

* Aura pulsation
* Lightning arcs
* Energy particles
* Speed lines
* Text impact entrances
* Radial shockwaves
* Floating rocks
* Scroll parallax
* Anime slash reveals
* Controlled screen shake
* Power-up sequences
* Manga panel transitions
* Cinematic light flashes

Avoid:

* Constant aggressive shaking
* Animations that block content
* Excessive motion on mobile
* Flashing that may cause discomfort
* Heavy effects that reduce performance
* Random animations without storytelling purpose

Respect `prefers-reduced-motion`. Provide a simplified experience with static backgrounds, subtle fades, and no screen shaking when reduced motion is enabled.

## Responsive Behaviour

Desktop:

* Cinematic full-screen compositions
* Deep parallax
* Large interactive universe layouts
* Cursor energy effects

Tablet:

* Simplified parallax
* Balanced section spacing
* Touch-friendly interactions

Mobile:

* Preserve the complete storytelling experience
* Reduce particles and background complexity
* Keep titles readable
* Use swipeable galleries
* Make every button at least 44px high
* Prevent horizontal overflow
* Avoid text overlapping animated effects
* Keep audio controls easily accessible

## Accessibility

Include:

* Semantic HTML
* Keyboard navigation
* Visible focus states
* Alt text for all images
* ARIA labels for sound and interactive controls
* Sufficient text contrast
* Reduced-motion support
* Escape-key support for modals
* No essential information communicated only through animation or sound
